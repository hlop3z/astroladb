## Define a User Table

Create `schemas/auth/user.js`:

```js
export default table({
  id: col.id(),
  email: col.email().unique(),
  username: col.username().unique(),
  password: col.password_hash(),
  is_active: col.flag(true),
}).timestamps();
```

## Generate Migration

```bash
alab new create_users
```

This creates a timestamped migration file in `migrations/`.

## Preview the SQL

```bash
alab migrate --dry-run
```

```sql
CREATE TABLE auth_users (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  email VARCHAR(255) NOT NULL UNIQUE,
  username VARCHAR(50) NOT NULL UNIQUE,
  password VARCHAR(255) NOT NULL,
  is_active BOOLEAN NOT NULL DEFAULT true,
  created_at TIMESTAMPTZ NOT NULL DEFAULT now(),
  updated_at TIMESTAMPTZ NOT NULL DEFAULT now()
);
```

## Apply Migration

```bash
alab migrate
```

## Export Types

Generate TypeScript types for your application:

```bash
alab export -f typescript
```

```typescript
// Generated: types/auth/user.ts
export interface User {
  id: string;
  email: string;
  username: string;
  is_active: boolean;
  created_at: Date;
  updated_at: Date;
}
```

:::note The `password` field is excluded from exports because
`col.password_hash()` is marked as hidden. :::
