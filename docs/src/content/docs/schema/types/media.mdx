---
title: Media Types
description: Content and media format types in AstrolaDB
---

Media types handle formatted content and visual data.

## Reference

| Type             | Expands To                       |
| ---------------- | -------------------------------- |
| `col.color()`    | `string(7)` - hex format #RRGGBB |
| `col.markdown()` | `text` + markdown format hint    |
| `col.html()`     | `text` + HTML format hint        |

## Example

```yaml
schema:
  posts:
    columns:
      id: col.id()
      content: col.markdown()
      rendered_html: col.html()
      theme_color: col.color()
```

## Color Format

Colors use 7-character hex format:

```
#FF5733  ✓
#fff     ✗ (use #FFFFFF)
rgb()    ✗
```

## Format Hints

The `markdown` and `html` format hints inform API clients and admin UIs to
render appropriate editors:

```yaml
# Client receives metadata:
{ "content": { "type": "text", "format": "markdown" } }
```
